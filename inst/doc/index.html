  • Citations. Package "QTLRel" was built on the R and C code that implemented the methodology described in Cheng et al. 2010. We ask users of the package to kindly cite the following papers:
    • Cheng R, Abney M, Palmer PP and Skol AD (2011). QTLRel: an R Package for Genome-wide Association Studies in which Relatedness is a Concern. BMC Genet. 12:66.
    • Cheng R, Lim JE, Samocha KE, Sokoloff G, Abney M, Skol AD and Palmer AA (2010). Genome-wide association studies and the problem of relatedness among advanced intercross lines and other highly recombinant populations. Genetics 185: 1033-1044.
